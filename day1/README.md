# Day 1 — Python Basics

## 📚 Concepts Learned

| Concept | Description |
|---|---|
| **Print Function** | Displays output to the screen using `print()` |
| **Variables** | Containers to store data values |
| **Variable Naming** | No spaces, cannot start with number, case sensitive |
| **Variable Assigning** | Assigning values to variables using `=` |
| **Single Line Comment** | Using `#` to add a note in code |
| **Multi Line Comment** | Using multiple `#` lines for longer notes |
| **Docstrings** | Using `""" """` to describe a program |
| **Indentation Error** | Raised when code spacing/indentation is wrong |
| **Syntax Error** | Raised when code grammar/syntax is wrong |
| **Name Error** | Raised when a variable is used before defining it |
| **Value Swapping** | Swapping two variables using temp variable and Python one-liner |

### 2. Value Swap Exercise
- Swap two variable values without typing the actual values
- Solved using both temp variable method and Python one-liner
- 📄 [value_swap_exercise.py](./value_swap_exercise.py)

**Test Case:**
```

After : glass1 = juice, glass2 = milk
```

---

## 💡 Key Takeaway
Python supports one-liner variable swapping: `a, b = b, a` — unique feature not available in most other languages!
