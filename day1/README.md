Day 1 — Python Basics
📚 Concepts Learned
1. Print Function - Displays output to the screen using print()
2. Variables - Containers to store data values
3. Variable Naming - No spaces, cannot start with number, case sensitive
4. Variable Assigning - Assigning values to variables using =
5. Single Line Comment - Using # to add a note in code
6. Multi Line Comment - Using multiple # lines for longer notes
7. Docstrings - Using """ """ to describe a program
8. Indentation Error - Raised when code spacing/indentation is wrong
9. Syntax Error - Raised when code grammar/syntax is wrong
10. Name Error - Raised when a variable is used before defining it
11. Value Swapping - Swapping two variables using temp variable and Python one-liner

2. Value Swap Exercise
- Swap two variable values without typing the actual values
- Solved using both temp variable method and Python one-liner

Test Case:

💡 Key Takeaway
Python supports one-liner variable swapping: a, b = b, a — unique feature not available in most other languages!
